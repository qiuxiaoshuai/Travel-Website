# 🧭 小帅的旅游探索网站 - PHP 版

欢迎来到 **小帅的旅游探索网站**！本项目是基于 PHP 开发的旅游信息平台，旨在为用户提供丰富、准确、便捷的旅游景点信息、推荐路线、游记分享等服务。适用于自由行爱好者、攻略达人及旅行初学者。

---

## 📌 项目简介

**网站特色功能包括：**

- 🏞️ **景点信息浏览**：支持省市级筛选，提供高清图片、地址、门票、开放时间等详尽信息。
- 🔍 **关键词搜索**：快速查找目的地景点。
- 🧳 **游记分享系统**：支持图文编辑与发布，展示他人真实旅游体验。
- 📍 **地图集成导航**：调用高德地图API，实现地理位置与路线指引。
- 👤 **用户系统（待开发）**：注册 / 登录 / 收藏 / 点赞 / 评论等互动功能。
